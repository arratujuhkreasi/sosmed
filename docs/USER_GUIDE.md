# User Guide - Horror Content App

## Pendahuluan

Selamat datang di Horror Content App! Aplikasi ini dirancang untuk membantu kreator konten YouTube dengan tema horror menemukan inspirasi, menganalisis performa, dan mengoptimalkan konten mereka.

## Memulai

### 1. Registrasi Akun

1. Buka aplikasi di browser
2. Klik tombol **Register** di navbar
3. Isi form registrasi:
   - Nama lengkap
   - Email
   - Nama channel YouTube (opsional)
   - Role (Creator/Visitor)
   - Password (minimal 6 karakter)
   - Konfirmasi password
4. Klik **Register**
5. Anda akan otomatis login dan diarahkan ke Dashboard

### 2. Login

1. Klik tombol **Login** di navbar
2. Masukkan email dan password
3. Klik **Login**
4. Anda akan diarahkan ke Dashboard

## Fitur Utama

### Dashboard

Dashboard adalah halaman utama setelah login yang menampilkan:

- **Statistik Konten**: Total konten, konten published, dan draft
- **Recent Content**: 5 konten terbaru Anda
- **Trending Topics**: 5 topik yang sedang trending

### Trends (Pencarian Inspirasi)

Halaman Trends membantu Anda menemukan topik yang sedang populer:

#### Melihat Trends

1. Klik **Trends** di navbar
2. Anda akan melihat daftar topik trending dengan:
   - Judul topik
   - Kategori (horror, creepypasta, urban-legend, dll)
   - Trending score
   - Keywords terkait
   - Popularity score

#### Filter by Category

1. Klik tombol kategori di bagian atas:
   - All (semua kategori)
   - Horror
   - Creepypasta
   - Urban Legend
   - Paranormal
   - True Crime
2. Trends akan difilter sesuai kategori yang dipilih

#### Search Trends

1. Ketik keyword di search box
2. Klik tombol **Search**
3. Hasil pencarian akan ditampilkan

### My Content (Manajemen Konten)

Halaman untuk mengelola semua konten Anda:

#### Melihat Konten

1. Klik **My Content** di navbar
2. Anda akan melihat semua konten dalam bentuk card
3. Setiap card menampilkan:
   - Judul
   - Deskripsi
   - Status (draft/published/archived)
   - Niche
   - Hook
   - Duration
   - Tags

#### Filter Konten

Klik tombol filter untuk melihat konten berdasarkan status:
- **All**: Semua konten
- **Published**: Konten yang sudah dipublish
- **Drafts**: Konten yang masih draft
- **Archived**: Konten yang diarsipkan

#### Membuat Konten Baru

1. Klik tombol **Create New Content**
2. Isi form:
   - **Title** (wajib): Judul konten
   - **Description** (wajib): Deskripsi lengkap konten
   - **Tags** (wajib): Tag dipisahkan dengan koma (contoh: horror, ghost, haunted)
   - **Niche** (opsional): Niche spesifik (contoh: Japanese Horror)
   - **Hook** (opsional): Hook menarik untuk konten
   - **Duration** (opsional): Durasi video dalam detik (30-120)
   - **Status**: Draft, Published, atau Archived
3. Klik **Create Content**
4. Konten berhasil dibuat dan Anda akan diarahkan ke halaman My Content

#### Edit Konten

1. Di halaman My Content, klik tombol **Edit** pada konten yang ingin diubah
2. Update informasi yang diperlukan
3. Klik **Update Content**

#### Hapus Konten

1. Di halaman My Content, klik tombol **Delete** pada konten
2. Konfirmasi penghapusan
3. Konten akan dihapus

#### View Detail Konten

1. Klik tombol **View** pada konten
2. Anda akan melihat detail lengkap konten
3. Termasuk data performa jika ada

### Profile

Mengelola profil pengguna:

1. Klik **Profile** di navbar
2. Anda dapat mengupdate:
   - Nama
   - Email
   - Nama channel
   - Password (jika ingin mengubah)
3. Klik **Update Profile**

## Tips Menggunakan Aplikasi

### Menemukan Niche Baru

1. Buka halaman **Trends**
2. Perhatikan topik dengan trending score tinggi
3. Lihat keywords yang digunakan
4. Cari kombinasi keywords yang belum banyak digunakan
5. Buat konten dengan angle yang berbeda

### Membuat Hook yang Menarik

1. Lihat contoh hooks dari trends yang populer
2. Gunakan pertanyaan yang memancing rasa penasaran
3. Mulai dengan sesuatu yang shocking atau unexpected
4. Jaga agar hook tetap singkat (1-2 kalimat)
5. Test beberapa variasi hook

### Mengoptimalkan Konten

1. Gunakan tags yang relevan dan populer
2. Pilih niche yang spesifik
3. Buat hook yang kuat di awal video
4. Perhatikan durasi optimal (60-90 detik untuk shorts)
5. Konsisten dengan tema dan style

### Menganalisis Performa

1. Catat data performa konten Anda:
   - Views
   - Likes
   - Comments
   - Shares
   - Watch time
2. Bandingkan konten yang perform baik vs kurang baik
3. Identifikasi pola:
   - Topik apa yang paling disukai?
   - Hook seperti apa yang efektif?
   - Durasi optimal?
4. Terapkan learning untuk konten berikutnya

## Best Practices

### Untuk Kreator Pemula

1. **Mulai dengan Niche Spesifik**
   - Jangan terlalu general
   - Fokus pada satu sub-kategori horror
   - Contoh: "Hantu Jepang" lebih baik dari "Horror"

2. **Riset Trends Secara Rutin**
   - Check trends minimal 2x seminggu
   - Catat topik yang konsisten trending
   - Identifikasi seasonal trends

3. **Konsisten dengan Upload Schedule**
   - Buat konten secara teratur
   - Gunakan status "draft" untuk konten yang sedang dikerjakan
   - Plan konten untuk 1-2 minggu ke depan

4. **Eksperimen dengan Format**
   - Test berbagai jenis hook
   - Coba durasi yang berbeda
   - Variasikan style storytelling

### Untuk Kreator Berpengalaman

1. **Analisis Data Mendalam**
   - Track performa setiap konten
   - Identifikasi pattern yang berhasil
   - Optimize berdasarkan data

2. **Diversifikasi Konten**
   - Jangan stuck di satu niche
   - Explore sub-kategori baru
   - Cross-pollinate ideas

3. **Stay Ahead of Trends**
   - Prediksi trends yang akan datang
   - Buat konten sebelum trend peak
   - Capitalize on early mover advantage

4. **Build Content Library**
   - Organize konten dengan baik
   - Reuse successful hooks
   - Create content templates

## Troubleshooting

### Tidak Bisa Login

- Pastikan email dan password benar
- Check caps lock
- Reset password jika lupa
- Clear browser cache

### Konten Tidak Tersimpan

- Pastikan semua field wajib terisi
- Check koneksi internet
- Refresh halaman dan coba lagi

### Trends Tidak Muncul

- Refresh halaman
- Check koneksi internet
- Coba filter kategori yang berbeda

### Error Saat Upload

- Check ukuran data
- Pastikan format input benar
- Coba logout dan login kembali
